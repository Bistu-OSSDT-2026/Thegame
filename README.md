# 🐍 贪吃蛇小游戏（the-game）

> 一个基于 HTML / CSS / JavaScript 的贪吃蛇游戏，支持键盘控制、分数统计、碰撞检测。  
> 本项目由 **5 人小组协作完成**，采用 GitHub 版本管理与分支工作流，实现从架构搭建、功能开发、自动化测试到正式发布的完整开发流程。

---

## 🚀 功能特性

- ✅ 蛇的移动与身体增长逻辑
- ✅ 食物随机刷新 + 吃到加分
- ✅ 碰撞检测（撞墙 / 撞自身则游戏结束）
- ✅ 键盘控制（上下左右转向）
- ✅ 实时分数显示 + 游戏重启
- ✅ 自动化 CI 检测代码质量
- ✅ 开源协议 & 贡献指南完善

---

## 🛠 技术栈

- **前端**：HTML / CSS / JavaScript（原生 DOM 操作）
- **版本管理**：Git + GitHub
- **协作工具**：
  - GitHub Issues：分配任务 & 记录 Bug
  - Pull Requests (PR)：代码审查机制
  - GitHub Actions：自动化 CI 检测
- **开源协议**：MIT License
